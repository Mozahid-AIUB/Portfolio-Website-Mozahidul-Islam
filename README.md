# Portfolio — Mozahidul Islam

Personal portfolio of **Mozahidul Islam** — Software Engineer · Developer · Researcher.

🔗 **Live:** [mozahidulislam.pro.bd](https://mozahidulislam.pro.bd)

## What's inside

- **Home** — two-column layout with scrollspy sidebar, cursor spotlight, particle field, shipping-log marquee, premium project showcases with 3D-tilt device mockups
- **Projects** — full filterable archive (Mobile / SaaS / Web / Desktop / Hardware / Database)
- **Research** — RAG, MCP tool protocols, EdTech engagement mechanics, programmatic visualization
- **Education, Resume, Contact** — with Web3Forms-powered contact form

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS v4 · Framer Motion

All content lives in typed data files under `data/` — adding a project means editing one file.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Deployed on Vercel. Contact form needs `NEXT_PUBLIC_WEB3FORMS_KEY` in `.env.local` (get a free key at [web3forms.com](https://web3forms.com)).
